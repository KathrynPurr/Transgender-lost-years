# E-Residency

Offitial site（官网）：https://e-resident.gov.ee/
## Introduction 介绍
https://e-estonia.com/

### E-resident card using 身份证使用：
1,核对卡片上的姓名 <br>

2,安装软件 https://www.id.ee/en/article/install-id-software/ <br>

3,将卡片插入读卡器背面的卡槽，读卡器连接电脑 USB 插口 <br>

4,登陆及管理密码 <br>
